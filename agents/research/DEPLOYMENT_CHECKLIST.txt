================================================================================
TROUBLESHOOTING
================================================================================

No results?
  → Check keywords (try simpler terms)
  → Increase depth level
  → Verify network connectivity
  → Run with --verbose for debugging

Want more sources?
  → Increase depth (1→2 or 2→3)
  → This increases search results by 2-3 sources

Output format issues?
  → Use --format pretty for readability
  → Use --output filename to save results
  → Verify JSON with: python3 -m json.tool < output.json

Need help?
  → python3 research.py --help
  → python3 test_research.py (run tests)
  → Check QUICKSTART.md for common issues
