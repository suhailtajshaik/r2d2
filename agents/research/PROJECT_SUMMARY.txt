PROJECT OVERVIEW
================================================================================

The Research Agent is a standalone Python application that performs autonomous
research queries using web search and content analysis. It returns structured
JSON output with key findings, source credibility analysis, and consensus
detection.

Key Characteristics:
  • Autonomous operation (no user interaction required)
  • Structured JSON output for easy integration
  • Source credibility scoring (0.0-1.0)
  • Contradiction detection
  • Consensus analysis
  • Configurable research depth (brief/standard/deep)
  • Keyword filtering for targeted research
  • Verbose logging for debugging

================================================================================
TROUBLESHOOTING GUIDE
================================================================================

ISSUE: No search results found
CAUSE: Network issue, keywords too specific
SOLUTION:
  - Try simpler keywords
  - Increase depth level
  - Check network connectivity
  - Run with --verbose for details

ISSUE: Slow performance
CAUSE: Depth 3 takes longer, network latency
SOLUTION:
  - Use depth 1 or 2 instead
  - Reduce keywords
  - Check network speed

ISSUE: JSON parsing error
CAUSE: Tool integration not enabled
SOLUTION:
  - Install requirements: pip install -r requirements.txt
  - Enable tools in wrapper (see INTEGRATION.md)
  - Check Python version (3.7+)

ISSUE: Memory usage high
CAUSE: Large content sizes, depth 3
SOLUTION:
  - Reduce depth level
  - Fetch smaller sources first

ISSUE: Output not detailed enough
CAUSE: Low depth level
SOLUTION:
  - Increase depth to 2 or 3
  - Add relevant keywords
  - Run longer research
