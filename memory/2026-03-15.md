## Meshtastic Research (March 15, 2026)
Suhail wants to integrate Meshtastic into his mesh network project later.

### What is Meshtastic?
- Open source, off-grid, decentralized mesh network
- Runs on cheap LoRa radio devices (no cell towers, no internet needed)
- Uses LoRa (Long Range) radio protocol — unlicensed, works worldwide
- Record range: 331km
- Encrypted, decentralized — no dedicated router required
- Devices rebroadcast messages they receive → true mesh
- Each device pairs with ONE phone at a time (via Bluetooth/WiFi/USB)
- Has meshtastic-cli for programmatic access
- 100% open source, community driven

### Key features
- Text messages across the mesh
- Optional GPS location sharing
- Excellent battery life (low power LoRa radios)
- Works without any infrastructure
